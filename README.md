# VSC-Productivity

## Terminal

```
ctrl + ` = Open terminal / Remove terminal
ctrl + maj + ` = Create new terminal
ctrl + maj + \ = Focus terminal tabs view
ctrl + PageUp/PageDown = Move to next terminal
F2 = Rename
```

## Code

```
ctrl + ← → = Skip word blocks
maj + alt + ↓ = Duplicate line/block select
alt + ↓↑ = Move line/block select
ctrl + maj + enter = New line up
ctrl + enter = New line down
ctrl + o = Go to declaration
ctrl + d = Select word focus
maj + ctrl + ← → = Select word blocks
ctrl + maj + e = focus explorer
ctrl + shift + k = Delet line
```

## Navigate

```
ctrl + maj + p = Ordering pallet
ctrl + maj + o = Go to symbol
ctrl + g = Go to line <number>
ctrl + b = Open / Remove sidebar
ctrl + \ = Split screen
ctrl + <number> = switch split screen
alt + <number> = switch file order
ctrl + PageUp/PageDown = switch file next/previous
ctrl + p = Select file
ctrl + w = remove focus file
ctrl + w + k = remove all file
ctrl + n = New file
ctrl + maj + n = New folder
ctrl + ↓↑ = Scroll file
```
